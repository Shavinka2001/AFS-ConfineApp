## ğŸ¯ Key Consolidation Logic

```javascript
// Pseudo-code for consolidation

Map<groupKey, groupData> = new Map()

for each workOrder in workOrders:
  // Create unique key
  key = workOrder.building + 
        workOrder.locationDescription + 
        workOrder.confinedSpaceDescription
  
  if Map.has(key):
    // Add to existing group
    group = Map.get(key)
    group.count++
    group.workOrderIds.push(workOrder.workOrderId)
    group.orders.push(workOrder)
  else:
    // Create new group
    Map.set(key, {
      building: workOrder.building,
      location: workOrder.locationDescription,
      space: workOrder.confinedSpaceDescription,
      count: 1,
      workOrderIds: [workOrder.workOrderId],
      orders: [workOrder]
    })

// Convert Map to Array
consolidatedGroups = Array.from(Map.values())

return consolidatedGroups
```
